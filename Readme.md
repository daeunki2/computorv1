# 프로젝트 개요

이 프로젝트는 간단한 다항식(polynomial) 방정식 풀이 프로그램을 만드는 과제이다.

기본 :
1차 방정식 혹은 2차 방적식.
미지수는 1개. 

핵심 목표:

- 문자열 형태의 방정식을 입력받는다
- 식을 정리(reduced form)한다
- 차수(degree)를 계산한다
- 0차 / 1차 / 2차 방정식을 해결한다
- 판별식(discriminant)에 따라 결과를 출력한다

---

# 허용 범위

- 2차 이하 방정식만 해결
- 3차 이상은 해결하지 않음
- 허수(complex) 해 처리 필요
- 언어 자유 선택 가능
- 직접 구현하지 않은 수학 함수 사용 금지
